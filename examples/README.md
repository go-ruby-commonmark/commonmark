# Ruby examples

Pure-Ruby examples for the `commonmark` library as provided by
[go-embedded-ruby](https://github.com/go-embedded-ruby/ruby) (rbgo). Run them
with the `rbgo` interpreter:

```sh
rbgo examples/commonmark_usage.rb
```

| File | Shows |
| --- | --- |
| [`commonmark_usage.rb`](commonmark_usage.rb) | `Commonmark.to_html` and `String#to_html`. |

Each example is executed as-is under rbgo (`require "commonmark"`).
